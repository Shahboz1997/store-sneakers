# store-sneakers
Описание проекта: Аналог интерфейса Ozon
Данный проект представляет собой аналог интерфейса российской e-commerce платформы Ozon. Реализованы ключевые элементы и основной функционал для демонстрации работы фронтенд-части торговой площадки.
Ключевые особенности проекта:
Меню навигации: Полноценная система навигации по разделам.
Рабочий поисковик: Функциональная строка поиска товаров.
Система фильтров: Детальная фильтрация товаров (по категориям, цвету, стоимости, размеру обуви), реализованная в левой и правой колонках.
Функция сортировки: Возможность сортировки результатов по различным параметрам (рейтинг, цена по возрастанию/убыванию, распродажа, новинки).
Навигация по страницам: Переходы между различными страницами.
Инструкции для локального запуска
Для запуска проекта на вашем локальном компьютере выполните следующие шаги в терминале или командной строке, находясь в корневой директории проекта:
1. npm i
2. npm run start
